Primary Care PTSD screen for DSM-5 (PC-PTSD-5) is a self-report screening tool used in primary care settings to identify individuals with probable PTSD.

Purpose: Designed for quick screening of trauma exposure and PTSD symptoms in time-limited primary care environments.

Format:

  • Consists of 6 yes/no items

  • If an individual screens positive for PTSD, further assessment is necessary. Ideally, this follow-up assessment should be done using a structured interview, such as the Clinician-Administered PTSD Scale for DSM-5 (CAPS-5). However, if this is unavailable, additional assessment using a validated self-report measure, such as the PTSD Checklist for DSM-5 (PCL-5), is recommended.
